Getting Started

Prerequisites

  • Go 1.23.8 or later
  • Access to Layer 8 ecosystem repositories
  • Git for version control

Quick Start

# Clone the repository git clone https://github.com/saichler/l8test.git cd l8test/go # Run the test suite ./test.sh # Or run tests manually go mod init GOPROXY=direct GOPRIVATE=github.com go mod tidy go mod vendor go test -tags=unit -v -coverpkg=./infra/... -coverprofile=cover.html ./...

Configuration Options

  • Node Count: Default 4 nodes in test topology
  • Port Ranges: [20000, 30000, 40000] for service allocation
  • Log Levels: Configurable (Trace, Debug, Info, etc.)
  • Replication: 2 replicas for distributed services
